# Bookr

A Django project based on the [Web Development with Django](https://subscription.packtpub.com/book/web-development/9781803230603) 

The documentation is built using Sphinx and ReST. It is built on every git push using
ReadTheDocs (https://readthedocs.org/projects/bookr/) and the documentation
is visible at https://bookr.readthedocs.io/en/latest/.
